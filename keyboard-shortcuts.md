# Keyboard Shortcuts

## Tabs

| Shortcut | Action |
|----------|--------|
| Super+T | New tab |
| Super+W | Close pane (or tab if last pane) |
| Super+Shift+[ | Previous tab |
| Super+Shift+] | Next tab |
| Super+1..9 | Switch to tab N |
| Super+Shift+R | Rename tab |
| Super+0 | Show all terminals (grid overview) |

## Splits

| Shortcut | Action |
|----------|--------|
| Super+D | Vertical split (within focused pane) |
| Super+Shift+D | Horizontal split (within focused pane) |
| Super+E | Vertical split at root (full-height column) |
| Super+Shift+E | Horizontal split at root (full-width row) |
| Super+Alt+Arrows | Navigate between panes |
| Super+Shift+Arrows | Swap focused pane with neighbor |
| Super+Ctrl+Arrows | Resize splits |

## Projects & Windows

| Shortcut | Action |
|----------|--------|
| Super+Alt+Shift+Left/Right | Move tab to prev/next project |
| Super+N | New window |
| Super+Shift+T | Detach tab to new window |
| Super+Shift+M | Merge all tabs into another window |
| Super+Q | Close window |
| Super+Alt+Q | Kill window (force) |

## Editing

| Shortcut | Action |
|----------|--------|
| Ctrl+Shift+C | Copy selection |
| Ctrl+Shift+V | Paste |
| Super+C / Super+V | Copy / Paste (if Super key works) |
| Super+F | Toggle search filter |
| Super+K | Clear scrollback and screen |
| Super+Backspace | Kill line (Ctrl+U) |
| Super+Left | Beginning of line (Home) |
| Super+Right | End of line (End) |
| Alt+Left | Word back |
| Alt+Right | Word forward |
| Shift+Enter | Newline without executing |
| Shift+Tab | Backtab (CSI Z) |
| Ctrl+A..Z | Send control character |

## System

| Shortcut | Action |
|----------|--------|
| F1 | Toggle help overlay |
| F2 | Save session |
| F11 | Toggle fullscreen |

## Mouse

| Action | Effect |
|--------|--------|
| Click + drag | Text selection |
| Right-click | Context menu (Copy / Paste) |
| Right-click project bar | Rename project |
| Drag tab | Reorder / move tab between projects |
| Scroll wheel | Scroll terminal history |

## Special Keys

| Key | Action |
|-----|--------|
| Arrow keys | Cursor movement (DECCKM-aware) |
| Home / End | Beginning / end of line |
| Page Up / Page Down | Scroll |
| Insert | Insert mode toggle |
| Delete | Forward delete |
| F2..F4 | Function keys (VT220) |
